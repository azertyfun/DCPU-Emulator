About DCPU-Toolchain

DCPU-Toolchain is an emulator, a debugger and an assembler for the techcompliant DCPU (formerly notch's in 0x10^c).

Supported hardware

  • DCPU;
  • CPU-Control (IACM);
  • Clock;
  • Keyboard;
  • LEM1802;
  • EDC;
  • M35FD.

Planned hardware

Everything planned by Paul.

How to build

  • Import with IntelliJ idea;
  • Install org.lwjgl:lwjgl:3.0.0b from Maven;
  • Download and add the 3.0.0b LWJGL natives from the LWJGL downloads page;
  • Enjoy!

Usage

Using the launcher

Simply download and run the launcher (also on github). It works as an alternative to the command line. It also saves your configuration for the next launch.

Using the command line

  • To run a program: java -jar DCPU-Toolchain.jar run <file> [--assemble] [--debugger] [--little-endian] [--clock] [--keyboard] [--lem1802] [--edc] [--M35FD=/path/to/file] [--M525HD=/path/to/file]: Runs emulator for (binary format) with specified hardware. If the --assemble flag is set, first assembles the file in the system temp directory. Big-endian by default, unless the --little-endian switch is present. If no hardware is specified, runs with clock, keyboard and LEM1802. --debugger enables a debugger interface with run/pause/stop/step, an interface to view the RAM and registers.
  • To assemble a program: java -jar DCPU-Toolchain.jar assemble <input file> <output file> [--little-endian]: Assembles to . Big-endian by default, unless the --little-endian switch is present.
